| US029 | Realizar pago en plataforma web | Como usuario, quiero realizar un pago en la plataforma para completar una transacción de forma segura. | **Escenario 1: Pago completado correctamente** **Dado que** el usuario ha ingresado la información de pago correcta y ha seleccionado la opción de "Pagar", **Cuando** el usuario confirma la transacción, **Entonces** el sistema debe procesar el pago de forma segura, mostrar un mensaje de confirmación de pago exitoso, y enviar un recibo al correo electrónico del usuario con los detalles de la transacción.**Escenario 2: Error en el proceso de pago** **Dado que** el usuario intenta realizar un pago en la plataforma, **Cuando** el sistema detecta información incorrecta (por ejemplo, datos de tarjeta de crédito inválidos) o el pago es rechazado por la entidad bancaria, **Entonces** el sistema debe mostrar un mensaje de error indicando la razón del fallo, y proporcionar instrucciones para corregir la información o intentar con otro método de pago. | EPIC007 |
